# Second Brain CLI
Overview:
A modular terminal-based personal knowledge system built with Python.

# Features:
- Create notes
- Search notes
- Tag system
- Favorites
- Flashcards
- Study mode
- SQLite persistence

# Tech Stack
- Python
- JSON
- SQLite
- Git
- Project Structure:
run.py
AI_Layer/ ai_config.py ai.py
Core_Features/ main.py models.py storage.py notes.py ui.py config.py
Storage/ data.json

# Goals
This project exists to:
- improve software engineering skills
- practice architecture and modularization
- learn persistence systems
- build a professional portfolio project
